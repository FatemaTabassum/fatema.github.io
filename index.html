PhD Candidate • AI for Software Security • ML Engineer

Building reliable AI systems for code, security, and real-world impact.

I am a PhD Candidate in Computer Science at Florida State University. My work focuses on machine learning for source code, including code representation learning, language models for code, and program-analysis-guided reasoning. I apply these methods to software engineering and security problems, especially source-code vulnerability detection in C/C++ programs. My broader interests include NLP for code, reliable AI systems, retrieval-augmented reasoning, and data-driven software engineering.
